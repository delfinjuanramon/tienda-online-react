# Sprint 2 – Flujo básico de compra

## Objetivo del Sprint
Desarrollar las funcionalidades relacionadas con la exploración del catálogo y el inicio del flujo de compra del sistema.

## Historias de Usuario desarrolladas
- HU-04 Visualización del catálogo de productos
- HU-05 Visualización del detalle de producto
- HU-06 Implementación básica del carrito de compras
- HU-07 Inicio del proceso de checkout

## Avances del Sprint
Durante este Sprint se implementó:

- Listado de productos mediante datos simulados
- Navegación hacia el detalle de producto
- Implementación inicial del carrito en versión simplificada
- Estructura básica del flujo de compra

## Ajustes realizados
Se detectó que la complejidad del carrito y del proceso de checkout requería dividir su implementación en varias iteraciones, por lo que parte de estas funcionalidades se programaron para el Sprint 3.

## Resultado del Sprint
Se logró establecer el flujo inicial de compra del sistema permitiendo al usuario explorar productos e iniciar el proceso de adquisición.
