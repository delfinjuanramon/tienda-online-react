# Sprint 1 – Navegación y exploración inicial del catálogo

## Objetivo del Sprint
Construir la base técnica del sistema de tienda en línea permitiendo navegación entre páginas, búsqueda y visualización inicial de categorías de productos.

## Historias de Usuario implementadas
- HU-01: Como usuario quiero visualizar la página de inicio para conocer el sistema.
- HU-02: Como usuario quiero navegar entre páginas para explorar el sitio.
- HU-03: Como usuario quiero buscar categorías para encontrar productos más rápido.
- HU-04: Como usuario quiero visualizar categorías para conocer el catálogo.

## Funcionalidades desarrolladas
- Configuración del entorno React + Vite
- Estructura de componentes reutilizables
- Implementación del Navbar
- Navegación con React Router
- Componente de búsqueda
- Listado de categorías dinámico

## Resultado del Sprint
Se obtuvo una versión funcional inicial del sistema que permite navegación básica, búsqueda y exploración del catálogo.

Implementar autenticación de usuarios y visualización de productos.
